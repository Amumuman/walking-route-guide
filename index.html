사용법

기본 사용법

  1. 루트 선택 (한강변 추천)
  2. 일차 탭으로 오늘 구간 보기
  3. 지도에서 마커 탭 → 상세정보
  4. 아래 카드 탭 → 지도에서 위치 확인
  5. 📍 내 위치 → GPS로 현재 위치 표시

실제 걸을 때

1. 오늘 일차 열기
→ 📍 경유지·길찾기 섹션에서
🗺️ 오늘 전체 루트 안내 시작 버튼 탭!

2. 네이버 지도가 열리면
→ GPS를 켠 상태에서 누르면 현재 위치 → 남은 경유지 기준으로 네이버 지도를 엽니다
→ GPS가 꺼져 있으면 오늘 시작점 → 종점 기준으로 엽니다
→ 이 페이지 안 지도는 실제 선형이 있을 때만 선이 보이고, 없으면 마커/구간 버튼만 보이는 게 정상이에요!

3. 맛집/숙소 볼 때만
→ 이 가이드로 돌아와서 확인

일정 변경 시: 경유지별 안내→ 버튼으로 아무 지점이나 개별 안내 가능

알아두면 좋은 것들

식당·숙소·편의점 카드를 탭하면?
→ 지도가 그 위치로 이동하고 상세 정보가 뜹니다!

마커 팝업의 네이버지도 → 버튼?
→ 네이버 지도에서 해당 장소를 바로 검색합니다

지도 위 마커 설명
🚩 출발/도착 · 📍 경유지 · 🍽️ 식당 · 🏨 숙소 · 🏪 편의점 · 🚉 전철역

식당 태그 설명
아침 점심 저녁 = 추천 시간대
~12km = 출발지에서 거리

⚠️ 오늘 휴무! 가 뜨면?
→ 그 식당이 오늘 쉬는 날이라는 뜻. 다른 곳으로!

일정이 바뀔 때

더 걷거나 덜 걸을 수 있으니까:
→ 경유지 옆 안내→ 버튼으로 아무 지점이나 개별 안내 가능
→ 🛏️ 일정 변경 시 섹션에서 중간에 잘 수 있는 곳 (✅) / 없는 곳 (❌) 확인

SOS 긴급

오른쪽 위 SOS → 경찰(112)·소방(119) 원터치, 가까운 전철역, 루트 기준 병원, 주변 응급실/약국/편의점 검색, GPS 좌표
