사용법

기본 사용법

  1. 루트 선택 (한강변 추천)
  2. 일차 탭으로 오늘 구간 보기
  3. 지도에서 마커 탭 → 상세정보
  4. 아래 카드 탭 → 지도에서 위치 확인
  5. 📍 내 위치 → GPS로 현재 위치 표시
