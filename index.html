사용법 가이드

1단계: 루트 선택

첫 화면에서 한강변 루트(안전) 또는 국도 루트(최단)를 탭하세요.

2단계: 페이스 선택

기본 = 하루 평균 20km (6~7일)
여유 = 하루 평균 14km (8~9일)
위쪽 버튼으로 언제든 전환 가능!

3단계: 일차별 보기

  1. 상단 일차 탭을 탭하면 해당 날의 지도가 뜹니다
  2. 지도 위 마커를 탭하면 상세 정보(메뉴, 가격, 영업시간)가 표시됩니다
  3. 지도 아래로 스크롤하면 맛집·숙소·편의점·전철역 목록이 있습니다

4단계: 걸으면서 사용

  1. 하단 📍 내 위치 버튼을 탭 → GPS가 켜지고 파란 점으로 현재 위치 표시
  2. 다음 경유지 이름을 확인 → 네이버 지도에서 "도보 길찾기"로 검색
  3. 도착하면 이 가이드로 돌아와 다음 경유지 확인
💡 이 가이드 = 전체 정보 확인용
💡 네이버 지도 = 실시간 도보 내비용
두 개를 번갈아 쓰는 게 가장 편해요!

5단계: GPX 파일 (몰라도 됨!)

GPX↓ 버튼 = 오늘 루트를 파일로 저장하는 기능입니다.
솔직히 안 써도 전혀 문제없어요! 이 가이드 + 네이버 지도면 충분합니다.

혹시 관심 있으면: GPX 파일을 등산/걷기 앱(트랭글, OsmAnd 등)에 불러오면 루트가 지도에 선으로 표시됩니다. 자세한 건 같이 드린 GPX 사용법 파일 참고.

SOS 긴급 버튼

오른쪽 위 SOS 버튼 → 경찰(112)·소방(119) 원터치 전화, 가장 가까운 전철역, 내 GPS 좌표가 표시됩니다.
